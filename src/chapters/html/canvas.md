
## Element `canvas`

The most exciting feature, HTML5 <canvas> tag allows us to render 2D shapes or graphics on web page with help of JavaScript. Moreover, weâ€™re able to create an animation using HTML5 Canvas.
The HTML below shows a simple HTML5 Canvas declaration and use JavaScript to draw a blue rectangle on it.

```html
<canvas id="myCanvas" width="200" height="200"></canvas>
<script>
   var c=document.getElementById("myCanvas");
   var ctx=c.getContext("2d");
   ctx.fillStyle="#0000FF";
   ctx.fillRect(0,0,150,150);
</script>
```
